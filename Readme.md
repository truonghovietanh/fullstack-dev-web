## day 1: html

- DOCTYPE là dạng DOC và phải luôn tuân thủ lệnh <!DOCTYPE html>
- các tag phải trong dấu <>
  thẻ mở <....>
  thẻ đóng </....>

- thẻ html sẽ chứa những thẻ sau

* <html>
      <head>
          <title> (bên trong head)
          <meta charset="utf-8" /> (thẻ này là hỗ trợ tiếng việt)
      <body> (bên ngoài head)
